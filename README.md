# Game Engine from Scratch
A very simplistic game engine implemented in Java, including Physics, Audio, Input, Rendering...

## Prerequisite

1. Java 8 and above
2. Intellij

## Code Style

1. Class structure
```
static fields
static methods
non-static fields
non-static methods
getters and setters
static classes
non-static methods
```

2. Add `@NonNull` annotations
